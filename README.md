# Word Frequency Report Service

Сервис для анализа частотности словоформ в текстовых файлах с поддержкой больших объемов данных и параллельной обработки.

## Описание тестового задания

Реализован REST API сервис, который принимает текстовый файл и возвращает Excel-отчет со статистикой словоформ. Сервис обрабатывает файлы любого размера (от килобайт до гигабайтов) и устойчив к параллельным запросам нескольких пользователей.

### Основные требования (из ТЗ)
- Прием текстового файла через HTTP
- Лемматизация слов (приведение к начальной форме: "житель" и "жителем" → одно слово)
- Подсчет статистики:
  - Общее количество словоформы во всем документе
  - Количество словоформы в каждой строке документа
- Генерация Excel-отчета с тремя колонками
- Обработка больших файлов без блокировки системы
- Устойчивость к параллельным запросам

## Архитектура проекта

Проект построен на принципах **DDD (Domain-Driven Design)** с четким разделением на слои:


### Ключевые архитектурные решения

| Решение | Обоснование |
|---------|-------------|
| **Чистая архитектура** | Инверсия зависимостей через абстракции, легкость тестирования и замены компонентов |
| **Потоковая обработка** | Файл читается чанками по 1 МБ, не загружается целиком в память |
| **Пакетная обработка** | Строки группируются в пакеты по 1000 для эффективной обработки |
| **Параллелизм** | ThreadPoolExecutor для обработки пакетов без блокировки asyncio |
| **Асинхронность** | FastAPI + aiofiles для неблокирующих I/O операций |
| **Временные файлы** | Автоматическая очистка после обработки |

## API Endpoints

### POST `/public/report/export`

Загружает текстовый файл и возвращает Excel-отчет со статистикой.

**Request:**
- `file` (multipart/form-data) - текстовый файл (.txt или .text)

**Response:**
- Excel файл (xlsx) с тремя колонками:
  1. Словоформа (лемматизированная)
  2. Количество во всем документе
  3. Количество в каждой строке (строка через запятую)

**Ограничения:**
- Максимальный размер файла: 3 ГБ
- Поддерживаемые форматы: .txt, .text

**Пример запроса:**
```bash
curl -X POST "http://localhost:8000/public/report/export" \
  -F "file=@document.txt" \
  --output report.xlsx
```
